Rounded Corners

Esta propiedad define el radio de las esquinas de un elemento

Sintaxis:
etiqueta-a-aplicar-estilo{
border-radius:(15px; se aplica sobre los cuatro por igual)
(15px 25px; el primer valor se aplica al lado superior izquierda e inferior derecha y el segundo valor al contrario )
(15px 25px 35px; el primer valor se aplica al lado superior izquierda el segundo al dado superior derecho e inferior izquierda y el tercero inferior derecha)
(15px 25px 35px 40px; los valores apican en sentido a las agujas del reloj empezando por la parte superior izquierda)
}
border-radius:valor;
border-top-left-radius:valor;
border-top-right-radius:valor;
border-bottom-right-radius:valor;
border-bottom-left-radius:valor;

Border Images

Se utiliza una imagen para ser usada como borde

Sintaxis:
etiqueta-a-aplicar-estilo{
border:(sin esto no se puede poner el borde);
border-image:(url de donde se situa la imagen) value round/strech ;
} border-image-source: url(ruta del archivo) ;
border-image-slice: corte ;
border-image-width: ancho ;
border-image-outset: salida por fuera del borde ;
border-image-repeat: queremos que se repita el borde ;

Gradient

Transiciones de color

Sintaxis:

background-image: linear-gradient(to(y la direccion o angulo que quiers que tome),colores por los que quieres que pase);

Text Effects

En relación con cómo los textos se comportan

Sintaxis:
etiqueta-a-aplicar-estilo{
text-overflow: clip/ellipsis(cuando el texto no puede no mostrar nada o puntos suspensivos);

word-break: keep-all/break-all(reglas de rotura de palabra);
writing-mode: horizontal/vertical(posición en la cual se escribe el texto)+-tb(top-bottom) o rl(right-left);

Web Fonts

Buttons
